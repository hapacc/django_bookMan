# 图书管理系统

### 简介

  图书管理系统，前端使用bootstrap，由Django框架驱动。
  由于，仅用两天编写，难免出现bug~
  仅仅为一个练习实例。

### 使用

```
连接SQL
python manage.py makemigrations bookMain
python manage.py migrate 
python manage.py runserver
```

### 功能

实现简单登陆注册忘记密码功能

实现分页功能(默认10项一页)

实现信息管理功能

实现借阅超时罚金等功能
